
# 🧩 Sudoku Master

A fully interactive and modern Sudoku game built using **HTML, CSS, and JavaScript**.  
It features a true random puzzle generator, smooth animations, and an app-like user experience.

---

## 🚀 Features

- 🎲 True random Sudoku generation on every game
- 🎮 Two difficulty levels: Easy & Advanced
- ✍️ Real-time input validation
- ✅ Correct answers lock automatically
- ❌ Wrong inputs trigger shake + error highlight
- 🎨 Modern UI with smooth animations
- 🔦 Row, column & box highlighting
- 🎉 Win detection with congratulations popup
- 📱 Fully responsive design (mobile & desktop)
- 🌙 Dark mode support (if enabled in version v3)
- 🔊 Sound effects for interaction (optional upgrade)
- ⏱️ Timer system (optional upgrade)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid + Animations)
- JavaScript (Vanilla JS)
